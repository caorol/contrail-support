Overview

Contrail is an email management application that works across multiple Gmail accounts. It automatically filters and surfaces only important emails while marking the rest as read, significantly reducing the need to manually check email.

Key Features

  • Support for multiple Gmail accounts
  • Automatic prioritization of important emails
  • Customizable rules based on labels and sender addresses
  • Automatic marking of non-important emails as read
  • Daily scheduled processing
  • Reduced email checking workload
  • Continuous optimization using AI (e.g., Apple Intelligence)
