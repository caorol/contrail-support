Contrail プライバシーポリシー

Gmail向けスマートメール優先表示アプリ
English Version

取得する情報

  • Google APIを通じて取得する Gmail のメールデータ
  • メールの重要度判定に必要なメール本文およびメタデータ

利用目的

取得した情報は以下の目的のみに利用します。

  • ユーザーへのメール表示
  • メールの重要度判定および優先順位付け
  • 重要でないメールの自動既読化

Google APIから取得した情報について

Contrail は Gmail の機能提供のためにのみ Google API を利用します。 Google APIから取得した情報を販売、第三者提供、広告目的で利用することはありません。

第三者提供

ユーザーデータを第三者へ販売、共有、貸与、提供することはありません。

保存期間

  • メール本文は、ユーザーが既読にしてアプリ上で表示されなくなるまで保存します。
  • 重要度判定に利用する情報は、ユーザーがアプリを利用している期間保存します。

ユーザーの権利

ユーザーは自身のデータをいつでも削除できます。 アカウントの削除またはアプリの利用停止により、保存されたデータの削除を行うことができます。

Google API Services User Data Policy

Contrail は Google API Services User Data Policy および Limited Use 要件を遵守し、 Google APIから取得したデータを適切に取り扱います。
