Contrail Privacy Policy

Smart Email Prioritization for Gmail
日本語版はこちら

Information We Collect

  • Gmail email data accessed through Google APIs
  • Email content and metadata required for prioritization

How We Use Information

Retrieved Gmail data is used solely to:

  • Display emails to the user
  • Determine email importance and ranking
  • Automatically mark non-important emails as read

Google API Data Usage

Contrail accesses Gmail data through Google APIs only for providing the application's functionality. Data obtained through Google APIs is not sold, shared, or used for advertising purposes.

Data Sharing

Contrail does not share, sell, rent, or disclose user data to any third parties.

Data Retention

  • Email content is retained until the email is marked as read and no longer displayed by the application.
  • Information used to determine email importance may be retained while the user continues using the application.

User Rights

Users may delete their own data at any time by removing their account or discontinuing use of the application.

Google API Services User Data Policy

Contrail's use and transfer of information received from Google APIs adheres to the Google API Services User Data Policy, including the Limited Use requirements.
